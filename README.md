## Overview
IPU-Connect is a web-based complaint management system designed for university students to raise, track, and manage complaints digitally.

## Features
- User & Admin authentication
- Raise and track complaints
- Ticket status: Open, In Progress, Resolved
- Admin dashboard
- Restricted ticket deletion

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MySQL / MongoDB (update as applicable)

## Live Demo
https://ipu-connect.vercel.app

## How to Run Locally
1. Clone the repo  
   `git clone https://github.com/satyam-edu/IPU-Connect.git`
2. Install dependencies  
   `npm install`
3. Start server  
   `node server.js`
